# Weather App 🌦️

A simple Weather App built using Python and Streamlit.

## Features
- Search weather by city
- Real-time weather information
- Temperature
- Humidity
- Wind Speed
- Weather Conditions

## Technologies Used
- Python
- Streamlit
- Weather API
